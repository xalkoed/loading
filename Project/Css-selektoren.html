CSS Selectors

  1. Type Selector

    Selector: element

    This selector targets all elements of a specific type, such as <p> or <h1>.

    
<style>
p {
    color: blue;
}
</style>

  2. Class Selector

    Selector: .className

    This selector targets elements with a specific class attribute.

    
<style>
.box {
    border: 1px solid black;
}
</style>

  3. ID Selector

    Selector: #idName

    This selector targets a specific element with a unique ID.

    
<style>
#header {
    font-size: 24px;
}
</style>

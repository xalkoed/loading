Definition List of HTML Elements

<html>
The root element of an HTML document.
<head>
Contains metadata about the document, such as title and links to stylesheets.
<body>
The main content area of an HTML document.
<p>
Defines a paragraph.
<a>
Defines a hyperlink.

HTML Code Example


<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>Sample HTML</title>
</head>
<body>
    <h1>Hello, World!</h1>
    <p>This is a sample paragraph.</p>
</body>
</html>
